queryOptions란, queryKey와 queryFn을 여러곳에서 공유하면서, 한번에 관리할 수 있는 헬퍼입니다.

- 기본 사용법

```ts
import { queryOptions } from "@tanstack/react-query";

function groupOptions(id: number) {
  return queryOptions({
    queryKey: ["groups", id],
    queryFn: () => fetchGroups(id),
    staleTime: 5 * 1000,
  });
}
```

위와 같이 정의하고, 아래와 같은 방식으로 쉽게 사용 가능합니다.

```ts
// useQuery
useQuery(groupOptions(1));

// useSuspenseQuery
useSuspenseQuery(groupOptions(5));

// useQueries — 여러 쿼리를 병렬로 실행
useQueries({
  queries: [groupOptions(1), groupOptions(2)],
});

// prefetchQuery — 미리 데이터 가져오기
queryClient.prefetchQuery(groupOptions(23));

// setQueryData — 캐시에 직접 데이터 설정
queryClient.setQueryData(groupOptions(42).queryKey, newGroups);
```

---

- Infinite Query용 헬퍼는 특수한 쿼리로서, `infiniteQueryOptios`헬퍼로 관리될 수 있습니다.

---

Select

select는 queryOptions로 설정값을 고유하면서, 동시에 일부 컴포넌트에서만 특정한 옵션을 넣어 override하는 방법이다.

```ts
const query = useQuery({
  ...groupOptions(1),
  select: (data) => data.groupName,
});
```
