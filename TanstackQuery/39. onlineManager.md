# OnlineManager

## 핵심 요약

OnlineManager는 TanStack Query에서 네트워크 온라인/오프라인 상태를 관리하는 도구입니다. 기본적으로 온라인 상태로 시작하며, 브라우저의 네트워크 연결 상태 변화를 감지하여 쿼리 동작을 제어합니다.

---

## 개념 설명

### OnlineManager란?

OnlineManager는 TanStack Query 내에서 **온라인 상태를 관리**하는 매니저입니다. 이를 통해 기본 이벤트 리스너를 변경하거나 수동으로 온라인 상태를 변경할 수 있습니다.

### 기본 동작 방식

기본적으로 onlineManager는:
- **활성 네트워크 연결이 있다고 가정**합니다
- **window 객체의 `online`과 `offline` 이벤트**를 감지하여 상태 변화를 파악합니다

### 왜 navigator.onLine을 사용하지 않나요?

이전 버전에서는 `navigator.onLine`을 사용하여 네트워크 상태를 확인했습니다. 하지만 **Chromium 기반 브라우저**(Chrome, Edge 등)에서 **false negative(거짓 음성)** 문제가 많이 발생했습니다.

**false negative란?**
- 실제로는 인터넷이 연결되어 있는데, 오프라인이라고 잘못 판단하는 경우
- 이로 인해 Query들이 불필요하게 오프라인으로 표시되어 데이터를 가져오지 못하는 문제 발생

### 현재 해결 방법

이 문제를 해결하기 위해:
- 항상 `online: true`로 시작
- `online`과 `offline` 이벤트만 감지하여 상태 업데이트

**장점:**
- false negative(잘못된 오프라인 판단) 가능성이 줄어듦

**단점:**
- **Service Worker**를 통해 인터넷 연결 없이도 작동하는 오프라인 앱의 경우, **false positive(거짓 양성)** 발생 가능
  - false positive: 실제로는 오프라인인데 온라인이라고 판단하는 경우

## 사용 가능한 메서드

### 1. setEventListener

커스텀 이벤트 리스너를 설정할 수 있습니다. 주로 **React Native**나 **다른 환경**에서 네트워크 상태를 감지할 때 사용합니다.

**예시: React Native에서 사용**

```tsx
import NetInfo from '@react-native-community/netinfo'
import { onlineManager } from '@tanstack/react-query'

onlineManager.setEventListener((setOnline) => {
  return NetInfo.addEventListener((state) => {
    setOnline(!!state.isConnected)
  })
})
```

**설명:**
- `NetInfo`는 React Native에서 네트워크 연결 상태를 확인하는 라이브러리
- `setOnline` 함수를 호출하여 연결 상태를 TanStack Query에 알림
- `!!state.isConnected`: 연결 여부를 boolean으로 변환

---

### 2. subscribe

온라인 상태 변화를 **구독(subscribe)**할 수 있습니다. 상태가 변경될 때마다 콜백 함수가 실행됩니다.

**예시:**

```tsx
import { onlineManager } from '@tanstack/react-query'

const unsubscribe = onlineManager.subscribe((isOnline) => {
  console.log('isOnline', isOnline)
})
```

**설명:**
- `subscribe`는 구독 해제 함수(`unsubscribe`)를 반환
- 온라인 상태가 바뀔 때마다 콜백이 실행되어 현재 상태를 로그로 출력
- 더 이상 구독이 필요 없을 때 `unsubscribe()`를 호출하여 메모리 누수 방지

---

### 3. setOnline

**수동으로** 온라인 상태를 설정할 수 있습니다. 테스트나 특정 시나리오에서 강제로 상태를 변경할 때 유용합니다.

**예시:**

```tsx
import { onlineManager } from '@tanstack/react-query'

// 온라인으로 설정
onlineManager.setOnline(true)

// 오프라인으로 설정
onlineManager.setOnline(false)
```

**언제 사용하나요?**
- **테스트 코드**에서 오프라인 상황을 시뮬레이션할 때
- 특정 조건에서 강제로 네트워크 상태를 변경하고 싶을 때

---

### 4. isOnline

현재 온라인 상태를 **조회**할 수 있습니다.

**예시:**

```tsx
const isOnline = onlineManager.isOnline()
```

**설명:**
- 현재 온라인 상태를 `boolean`으로 반환
- 조건부 로직에서 사용 가능

## 핵심 요점 정리

| 개념 | 설명 |
|------|------|
| **OnlineManager** | TanStack Query의 온라인/오프라인 상태를 관리하는 매니저 |
| **기본 동작** | 항상 online: true로 시작하고, window의 online/offline 이벤트를 감지 |
| **navigator.onLine 사용 중단** | Chromium 브라우저에서 false negative 문제 발생으로 사용 안 함 |
| **setEventListener** | 커스텀 이벤트 리스너 설정 (React Native 등에서 활용) |
| **subscribe** | 온라인 상태 변화 구독 |
| **setOnline** | 수동으로 온라인 상태 설정 (테스트 등에서 활용) |
| **isOnline** | 현재 온라인 상태 조회 |

## 어려운 개념 풀이

### False Negative vs False Positive

| 용어 | 의미 | 예시 |
|------|------|------|
| **False Negative (거짓 음성)** | 실제로는 참인데 거짓이라고 판단 | 인터넷이 연결되어 있는데 오프라인이라고 판단 |
| **False Positive (거짓 양성)** | 실제로는 거짓인데 참이라고 판단 | 인터넷이 끊겼는데 온라인이라고 판단 |

### Service Worker란?

- 브라우저가 백그라운드에서 실행하는 스크립트
- 네트워크 요청을 가로채서 캐시된 데이터를 제공할 수 있음
- **인터넷 연결 없이도 앱이 작동**할 수 있게 만드는 기술 (PWA의 핵심)

## 실전 활용 예시

### React Native에서 네트워크 상태 감지

```tsx
import NetInfo from '@react-native-community/netinfo'
import { onlineManager } from '@tanstack/react-query'

// 앱 초기화 시 설정
onlineManager.setEventListener((setOnline) => {
  return NetInfo.addEventListener((state) => {
    setOnline(!!state.isConnected)
  })
})
```

### 테스트에서 오프라인 상황 시뮬레이션

```tsx
import { onlineManager } from '@tanstack/react-query'

// 테스트 시작 전 오프라인으로 설정
beforeEach(() => {
  onlineManager.setOnline(false)
})

// 테스트 종료 후 온라인으로 복원
afterEach(() => {
  onlineManager.setOnline(true)
})
```

### 네트워크 상태에 따른 UI 표시

```tsx
import { onlineManager } from '@tanstack/react-query'
import { useEffect, useState } from 'react'

function NetworkStatus() {
  const [isOnline, setIsOnline] = useState(onlineManager.isOnline())

  useEffect(() => {
    const unsubscribe = onlineManager.subscribe(setIsOnline)
    return unsubscribe
  }, [])

  return (
    <div>
      {isOnline ? '✅ 온라인' : '❌ 오프라인'}
    </div>
  )
}
```

## 마무리

OnlineManager는 TanStack Query가 네트워크 상태를 인식하고 적절하게 대응할 수 있도록 돕는 중요한 도구입니다. 특히 모바일 환경이나 네트워크가 불안정한 환경에서 사용자 경험을 개선하는 데 필수적입니다.
